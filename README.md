  # 🛍️ Mini Marketplace

A simple web application to buy, sell, and discover items in your community. Built with PHP & MySQL as a **Web Technology Project**.

## ✨ Features

- **User Registration & Login** — Create an account and sign in with email and password
- **Browse Items** — View all marketplace listings (no login required)
- **List Products** — Add items with title, description, price, and product image
- **Edit & Delete** — Manage your own listings
- **Image Uploads** — Support for JPG, PNG, GIF, and WebP
- **Admin Panel** — Admin users can manage all users and items
- **Responsive Design** — Works on desktop and mobile

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS
- **Database:** MySQL
- **Backend:** PHP

## 📋 Requirements

- PHP 7.4+
- MySQL 5.7+ / MariaDB
- Apache (XAMPP recommended)
